# 🏦 Banking Management System

## 📖 Description
This is a Banking Management System project developed using PHP and MySQL. 
It allows users to manage accounts, transfer money, and perform banking operations.

---

## 🚀 Features
- User Registration & Login
- Account Management
- Money Transfer
- Admin Dashboard
- CSP Panel

---

## 🛠️ Technologies Used
- PHP
- MySQL
- HTML, CSS
- XAMPP

---

## ⚙️ Installation
1. Download or clone the project
2. Move to XAMPP htdocs folder
3. Import database from `Database/database.sql`
4. Start Apache & MySQL
5. Open browser: http://localhost/bankingmg
